• WPS 설정
    • WPS 설정은 무선 설정을 초보자도 쉽게 할 수 있도록 도와 줍니다.
  • WPS 기본 설정
    • WPS 활성화
      • 활성화로 설정 시 WPS 버튼이 동작합니다.
      • 비활성화로 설정 시 WPS 버튼이 동작하지 않습니다.
    • WPS 연결 방법
      • WPS 버튼(PBC)을 이용한 방법
        • 공유기의 CPU/RUN LED가 켜진 상태로 유지될때까지 WPS버튼을 1~2초간 누릅니다.
        • 2분안에 무선 랜카드의 WPS버튼(PBC버튼)을 누릅니다.
        • 무선 랜카드와 무선공유기의 연결이 올바르게 수립되었는지 확인합니다.
      • PIN 방식으로 연결하는 방법
        • WPS연결하기의 "LAN Card의 PIN번호 입력하여 연결" 을 선택합니다.
        • LAN Card PIN 입력 칸에 랜카드의 PIN CODE를 입력합니다.
        • "WPS 연결" 버튼을 클릭합니다.
        • 랜카드 용 Utility 상에 있는 PIN을 사용하여 연결하는 동작을 수행합니다.
        • 무선 랜카드와 무선공유기의 연결이 올바르게 수립되었는지 확인합니다.
