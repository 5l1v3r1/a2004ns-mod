• 개 요
    • 멀티 브리지 기능을 이용하면, 다른 AP로의 브리지 또는 무선 WAN포트 연결이 가능합니다.
  • 사용 방법
    • 검색된 AP중 원하는 AP를 선택합니다.
    • [무선 브리지] 또는 [무선 WAN] 기능을 선택합니다.(아래 참조)
    • AP 선택 시, 인증 및 암호화 방법은 자동으로 설정됩니다.
    • 암호화된 AP로의 접속시에는 네트워크 암호를 입력합니다.
    • [적용] 버튼을 누릅니다.
  • 무선 동작 모드
    • 무선 브리지
      • 연결하려는 AP와 같은 IP서브넷으로 구성 시 사용됩니다.
      • 연결된 AP로의 무선 연결을 브리지로 사용합니다.
      • 무선 연결을 통과시 모든 소스 MAC주소는 브리지 맥주소로 변환됩니다.
      • IP프로토콜 이외의 프로토콜은 통신할 수 없습니다.
    • 무선 WAN
      • 연결된 AP로의 무선 연결을 WAN포트로 사용합니다.
      • [ 인터넷 연결 설정 ]을 통하여 무선WAN포트를 설정할 수 있습니다.
      • 기존의 유선 WAN포트를 사용할 수 없습니다.
